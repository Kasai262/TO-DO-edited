To-Do App

An upgraded To-Do application built with HTML, CSS and Vanilla JavaScript — no frameworks, no libraries.

Features


Add tasks with a due date
Edit task titles inline (Save / Cancel)
Delete tasks with a confirmation prompt
Mark tasks as completed / undo
Real-time search (case-insensitive)
Sort by due date — oldest or newest first
Filter tasks — All / Active / Completed
Live statistics — total, active and completed task count
Clear all completed tasks at once
Drag & Drop manual reordering (bonus)
Full localStorage persistence
Input validation — minimum 3 characters, title required, date required
Responsive design
