# To-Do App

Simple To-Do application made with HTML, CSS and Vanilla JavaScript. Started as a basic task manager, upgraded with new features as part of a school assignment.

## Features

- Add tasks with a due date
- Edit task title inline
- Delete tasks (with confirmation)
- Mark tasks as completed / undo
- Search tasks in real time (case-insensitive)
- Sort by due date — oldest or newest first
- Filter by All / Active / Completed
- Task statistics — total, active, completed
- Clear all completed tasks
- Drag & Drop reordering
- localStorage — tasks stay after refresh
- Input validation

## Technologies

- HTML5
- CSS3
- JavaScript
- localStorage

## How to run

Just open `index.html` in a browser, no setup needed.

## Validation

- Title can't be empty
- Minimum 3 characters
- Due date is required

## Notes

No frameworks or libraries used — only vanilla JS as required by the assignment.
